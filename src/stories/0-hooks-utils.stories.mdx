# ReactJS Hooks and Utilities

Custom hooks leverage the power of React hooks to add additional functionality to our React apps. <br/>Below you can find our most used ReactJS hooks and utilities.

### useOnClickOutside
````jsx harmony
import React, {useRef, useState} from 'react';
import {useOnClickOutside} from '@datorama/app-components';

const Comp = () => {
  const [open, setOpen] = useState(false);
  const ref = useRef(null);
  const handleClickOut = () => setOpen(false);
  
  // (ref, handler, isActive) 
  useOnClickOutside(ref, handleClickOut, open);
  
  return <Popup ref={ref} />;
};
````

### useWindowSize
````jsx harmony
import React from 'react';
import {useWindowSize} from '@datorama/app-components';

const Comp = () => {
  const {
    innerWidth,
    innerHeight,
    outerWidth,
    outerHeight
  } = useWindowSize();
  
  return <Page/>;
};
````

### Convert hex to rgba
````javascript
import {hexToRgba} from '@datorama/app-component';

// hex and opacity (0 - 100)
const color = hexToRgba('#ff0000', 50); // rgba(255, 0, 0, 0.5)
````

### Generate uuid
````javascript
import {uuid} from '@datorama/app-component';

const uniqueId = uuid();
````
