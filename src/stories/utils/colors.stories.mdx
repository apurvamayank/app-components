# Utils: Color utils

## Convert hex to rgba

````javascript
import { hexToRgba } from '@datorama/app-component';

// hex and opacity (0 - 100)
const color = hexToRgba('#ff0000', 50); // rgba(255, 0, 0, 0.5)
````
