## Pending Deprecation

The following components are pending deprecation and will be removed in next releases. Avoid using them in new features.

`Table`, `IllustratedMessage`, `Ellipsis`, `Draggable`, `Droppable`, `DragDropProvider`, `withToast`, `withSnackbar`, `ToastsProvider`

## 1.9.1
- Fixed `timeout` parameter in notifications config. When set to `Infinity`, the notification will not be cleared automatically
- Return `onClear(id)` & `onClearAll()` callbacks on `useToast()` & `useSnackbar()` hooks usage:

```
 const { addSnackbar, onClear, onClearAll } = useSnackbar();
```

- Return notification id on `addToast` & `addSnackbar` usage, allowing the user to clear a notification programatically:

```
const { addToast, onClear } = useToast();
const id = addToast(config);
...
onClear(id);
```
