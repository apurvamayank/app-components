# Flex-box Layout

A 12 column grid layout using flex-box
<br /><br />

## Auto width layout

```html
<Container>
  <Row>
    <Col />
    <Col />
    <Col />
  </Row>
</Container>
```

## Columns size

```html
<Container>
  <Row>
    <Col size={4}/>
    <Col size={6} />
    <Col size={2} />
  </Row>
</Container>
```

## Columns offset

```html
<Container>
  <Row>
    <Col offset={2} size={4}/>
    <Col size={6} />
  </Row>
</Container>
```

## Column distribution

```html
<Container justify="flex-end">
  <Row>
    <Col offset={2} size={4}/>
    <Col size={6} />
  </Row>
</Container>
```
